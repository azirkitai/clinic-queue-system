Mantap 🔥 Sekarang kita fokus pada isu utama:
👉 Paparan TV tak auto-resize dengan betul (layout lari bila resolusi TV berubah)

Masalah ni memang biasa berlaku bila layout dibuat dengan unit tetap (px) atau tanpa struktur responsive yang betul.
Mari aku tunjukkan cara paling stabil untuk pastikan paparan auto resize elok di TV, tak kira resolusi 💻📺👇

🧱 1. Guna viewport meta tag (WAJIB)

Dalam bahagian <head> HTML, pastikan ada ini:

<meta name="viewport" content="width=device-width, initial-scale=1.0">


👉 Ini memastikan browser pada TV atau Mac ikut skala sebenar device — tanpa ni, layout akan “zoom pelik” atau tak resize langsung.

📐 2. Elakkan unit px statik — guna unit vw, vh, % atau flex

Contoh layout yang SALAH:

.container {
  width: 1920px;
  height: 1080px;
}
.title {
  font-size: 40px;
}


Contoh layout yang BETUL (responsive):

.container {
  width: 100vw;   /* 100% lebar skrin */
  height: 100vh;  /* 100% tinggi skrin */
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
}

.title {
  font-size: 4vw; /* 4% dari lebar viewport */
}


📌 vw = viewport width → sangat sesuai untuk teks / elemen yang perlu scale ikut skrin TV besar
📌 vh = viewport height → untuk tinggi layout penuh skrin

🧰 3. Gunakan Flexbox / CSS Grid untuk Auto-Resize Struktur

Contoh paparan queue board simple:

<div class="screen">
  <h1 class="title">PAPARAN GILIRAN</h1>
  <div class="queue">
    <div class="counter">Kaunter 1</div>
    <div class="number">A102</div>
  </div>
</div>

.screen {
  width: 100vw;
  height: 100vh;
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  background: #111;
  color: white;
}

.title {
  font-size: 5vw;
  margin-bottom: 2vh;
}

.queue {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 2vw;
  width: 80vw;
  font-size: 4vw;
  text-align: center;
}


✅ Kelebihan:

Bila resolusi TV besar → elemen ikut skala automatik

Bila buka di Mac (simulate 1920×1080) → susun atur kekal kemas

Tak perlu manual resize setiap elemen

🪄 4. Tambah CSS @media jika nak kontrol design ikut saiz TV

Contoh: Kalau paparan TV 1366×768, boleh tweak sikit font size:

@media (max-width: 1400px) {
  .title {
    font-size: 6vw;
  }
}

🚀 5. (Optional) Auto fullscreen bila load

Kalau nak paparan betul-betul TV mode:

document.addEventListener('DOMContentLoaded', () => {
  if (document.documentElement.requestFullscreen) {
    document.documentElement.requestFullscreen();
  }
});


👉 Jadi tak perlu tekan F11 manual.
